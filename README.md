# Python To-Do List

A simple command-line To-Do List application built using Python.  
This project allows users to manage their daily tasks directly from the terminal or on an Android device using the **Coding Python app**.

## Features
- Add new tasks
- View all tasks
- Remove completed tasks
- Simple and beginner-friendly command-line interface

## Technologies Used
- Python 3
- Android app: Coding Python (for running the script on mobile)

## How to Run

1. Download or clone the repository
2. Open the terminal or Coding Python app
3. Navigate to the project folder
4. Run the program:

python Basic_to-do_list.py

## Example

Choose an option:
1. Add Task
2. View Tasks
3. Remove Task
4. Exit
